📖 How to Subscribe

📱 Google Calendar
  1. Open Google Calendar
  2. Click + next to "Other calendars" on the left sidebar
  3. Select "From URL"
  4. Paste the ICS URL above
  5. Click "Add calendar"
🍎 Apple Calendar (macOS / iOS)
  1. Open the Calendar app
  2. Go to File → New Calendar Subscription
  3. Paste the ICS URL above
  4. Click Subscribe
  5. Set auto-refresh interval (e.g., every hour)
📧 Microsoft Outlook
  1. Open Outlook Calendar
  2. Click Add calendar → From Internet
  3. Paste the ICS URL above
  4. Click OK
